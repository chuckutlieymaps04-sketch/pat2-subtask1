# Morse Code

# What is Morse Code?
Morse code is a way of sending messages using dots (.) and dashes (-). Each letter and number has its own pattern. It was used a long time ago before phones and the internet to communicate over long distances.

# History of Morse Code
Morse code was invented by Samuel Morse and Alfred Vail in the 1830s. It was mainly used with the telegraph machine to send messages through wires. It became very important for communication, especially in ships and during wars.

# How Morse Code Works
Morse code works by using:
- Dots (short signals)
- Dashes (long signals)

Each letter is made using a combination of dots and dashes. There is a small space between letters and a bigger space between words.

# Examples
Some examples of Morse code:

- A = .-
- B = -...
- C = -.-.

Words:
- HELLO = .... . .-.. .-.. ---
- SOS = ... --- ...
- CODE = -.-. --- -.. .

# Conclusion
Morse code was very useful in the past and helped people communicate over long distances. Even though we don’t use it much today, it is still important to learn about it.
